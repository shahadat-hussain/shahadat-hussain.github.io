# Dr. Shahadat Hussain - Academic Portfolio

A professional academic portfolio website built with modern web technologies, hosted on GitHub Pages.

## Features

- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Dark/Light mode toggle
- ✅ Publication filtering system
- ✅ Contact form
- ✅ Research metrics display
- ✅ Accessibility optimized
- ✅ SEO optimized
- ✅ Fast loading performance

## Quick Start

1. **Fork this repository** or create a new one with name `username.github.io`
2. **Enable GitHub Pages** in repository Settings → Pages
3. **Customize** the content in `index.html`
4. **Update** your personal information and publications
5. **Add** your profile picture to `/assets/images/profile.jpg`

## Customization

### Personal Information
Edit the following sections in `index.html`:
- Hero section (name, title, institution)
- Contact information
- Research interests
- Publications list
- Experience timeline

### Styling
Modify CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    --accent-color: #1abc9c;
    /* ... */
